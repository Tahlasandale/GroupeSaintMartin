# **App Name**: AuthZen

## Core Features:

- Email/Password Login: Allow users to log in using their email address and password.
- Email/Password Signup: Enable users to create new accounts using their email address and password.
- Google Authentication: Implement 'Sign in with Google' for quick and easy account creation and login.
- Form Toggle: Provide a clear and intuitive toggle to switch between the login and signup forms.

## Style Guidelines:

- Primary color: Muted Green (#90EE90), offering a subtle, secure feel.
- Background color: Very light gray (#f0f0f0), creating a clean and minimalist backdrop.
- Accent color: Deep Green (#228B22) for emphasis on interactive elements like buttons.
- Font: 'Inter', sans-serif, for a clean, modern, readable style in both body and headlines.
- Simple, line-based icons for form elements (e.g., email, password) to maintain a minimalist aesthetic.
- Clean, single-column layout for easy navigation, suitable for mobile and desktop. Forms should be clearly delineated with sufficient spacing for a minimalist look.
- Subtle fade-in and slide-in animations on form elements for smooth transitions, enhancing the user experience.
